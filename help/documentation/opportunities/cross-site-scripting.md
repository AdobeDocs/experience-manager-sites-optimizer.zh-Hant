# 跨網站指令碼機會

![跨網站機會](./assets/cross-site-scripting/hero.png){align="center"}

跨網站指令碼機會會識別您網站程式碼中的漏洞。 接著，它會修正攻擊者可能利用惡意指令碼插入其他使用者所檢視之網頁的問題。 這些指令碼可以竊取敏感資訊 (例如工作階段 Cookie)，或代表使用者執行操作 (例如變更使用者的密碼)。

* **重現連結** - 漏洞發現所在頁面的連結。
