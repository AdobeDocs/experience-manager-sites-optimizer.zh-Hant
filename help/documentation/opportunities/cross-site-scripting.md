# 跨網站指令碼機會

![跨網站機會](./assets/cross-site-scripting/hero.png){align="center"}

跨網站指令碼機會會識別並修正網站程式碼中的漏洞，攻擊者可能會利用這些漏洞將惡意指令碼插入其他使用者檢視的網頁中。 這些指令碼可能會竊取敏感資訊（例如工作階段Cookie），或代表使用者執行動作，例如變更使用者的密碼。
