# 預檢連結機會

![預檢連結機會](./assets/links/hero.png){align="center"}

Sites Optimizer中的「預檢連結」機會可協助您識別及解決與網站上使用連結相關的問題。 此機會著重於評估網站的連結結構，並提供可行的改善建議。
