# 預檢可讀性機會

![預檢可讀性機會](./assets/readability/hero.png){align="center"}

Sites Optimizer 中的預檢可讀性機會可協助您識別和解決網站內容可讀性的相關問題。 此機會聚焦於評估您網站的內容結構，並提供可操作的改善建議。

預檢可讀性機會提供多種語言版本，包括：

* 英文
* 法文
* 西班牙文
* 義大利文
* 德文
* 荷蘭文

多語言版本確保團隊可以針對廣泛客群評估並最佳化內容易讀性。
