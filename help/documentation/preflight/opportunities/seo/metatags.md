# 中繼資料稽核

**中繼資料**&#x200B;稽核會檢閱您頁面上的中繼資料標籤，例如頁面標題和中繼資料說明。 格式正確的中繼資料可協助搜尋引擎瞭解您的內容，並改善您的頁面在搜尋結果中的顯示方式。 稽核會標示遺失、重複或超出建議長度的標籤。

當稽核發現商機時，每個商機都會識別受影響的標籤及其目前值，並可能包含AI產生的改善值建議。 若要瞭解如何檢閱和解決商機，請參閱[預檢中的稽核結果](../../audit-results.md)。
