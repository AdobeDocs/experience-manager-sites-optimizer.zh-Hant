# 標準稽核

**Canonical**&#x200B;稽核會檢閱您頁面上的標準連結。 正確的規範連結會告知搜尋引擎，哪個URL是頁面的權威版本，這有助於避免重複內容問題。 稽核會標示缺少或不正確的標準連結。

當稽核發現商機時，每個商機都會說明問題和建議的變更。 若要瞭解如何檢閱和解決商機，請參閱[預檢中的稽核結果](../../audit-results.md)。
