# 連結稽核

**連結**&#x200B;稽核會檢閱您頁面上的連結。 中斷或不安全的連結會損害使用者體驗，並降低搜尋引擎抓取您網站的效率。 稽核會標示需要注意的連結，以便在發佈前修正這些連結。

當稽核發現商機時，每個商機都會識別受影響的連結，並描述建議的變更。 若要瞭解如何檢閱和解決商機，請參閱[預檢中的稽核結果](../../audit-results.md)。
