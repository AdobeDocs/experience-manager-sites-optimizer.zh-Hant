# 標題稽核

**Headings**&#x200B;稽核會檢閱頁面的標題結構。 清晰、有序的標題階層有助於讀者和搜尋引擎瞭解內容的組織方式。 稽核會標示問題，例如略過層次或顯示順序不正確的標題。

當稽核發現商機時，每個商機都會識別受影響的標題，並描述建議的變更。 若要瞭解如何檢閱和解決商機，請參閱[預檢中的稽核結果](../../audit-results.md)。
