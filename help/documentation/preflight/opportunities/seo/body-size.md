# 內文大小稽核

**內文大小**&#x200B;稽核會檢閱您頁面上的內文內容量。 內容很少的頁面可能對讀者不太實用，並且可能在搜尋結果中排名不佳。 稽核會標籤似乎內容不足的頁面。

當稽核發現商機時，每個商機都會說明問題和建議的變更。 若要瞭解如何檢閱和解決商機，請參閱[預檢中的稽核結果](../../audit-results.md)。
