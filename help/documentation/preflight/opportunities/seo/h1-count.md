# H1計數稽核

**H1計數**&#x200B;稽核會檢閱您頁面上的H1標題數量。 通常頁面應該有單一描述性H1以摘要其內容。 缺少或多個H1標題可能會削弱頁面的結構及其搜尋引擎最佳化。

當稽核發現商機時，每個商機都會說明問題和建議的變更。 若要瞭解如何檢閱和解決商機，請參閱[預檢中的稽核結果](../../audit-results.md)。
