# 可讀性稽核

**可讀性**&#x200B;稽核會檢閱您的頁面內容閱讀的容易程度。 清晰、結構良好的內容可讓讀者持續參與，並有助於更廣大的受眾瞭解您的訊息。 稽核會評估內容，並提供可行的改善建議。

可讀性稽核支援多種語言的內容，包括：

* 英文
* 法文
* 西班牙文
* 義大利文
* 德文
* 荷蘭文

當稽核發現商機時，每個商機都會說明問題和建議的變更。 若要瞭解如何檢閱和解決商機，請參閱[預檢中的稽核結果](../../audit-results.md)。
