# Lorem ipsum稽核

**Lorem ipsum**&#x200B;稽核會偵測頁面內容中留下的預留位置文字。 對讀者來說，發佈含有預留位置文字的頁面似乎尚未完成，而且可由搜尋引擎編制索引。 稽核會標籤所有預留位置文字，讓您在發佈前可以取代為真實內容。

當稽核發現商機時，每個商機都會識別預留位置文字出現的位置。 若要瞭解如何檢閱和解決商機，請參閱[預檢中的稽核結果](../../audit-results.md)。
