# 預檢無障礙機會

![預檢無障礙機會](./assets/accessibility/hero.png){align="center"}

Sites Optimizer中的「預檢協助工具」機會可協助您識別並解決網站上的協助工具問題，以確保所有人士（包括殘疾人士）都能使用。 此機會聚焦於評估網站無障礙標準的合規性，並提供可操作的改善建議。
