# 協助工具稽核

![預檢整備儀表板中的協助工具稽核](./assets/accessibility/hero.png){align="center"}

Preflight中的&#x200B;**協助工具**&#x200B;稽核會根據網頁內容協助工具准則(WCAG)評估您的頁面，以協助確保每個人（包括殘障人士）都可以使用頁面。 當您執行「預檢」稽核時，它會根據這些協助工具標準評估頁面，並將它發現的問題分組為您可以在發佈之前複查和解決的機會。

在整備儀表板中，展開&#x200B;**協助工具**&#x200B;類別，以檢視稽核是否通過以及找到的機會數目。 選取稽核以開啟其詳細資訊頁面並處理商機。 每個機會都會說明協助工具問題、其嚴重性、相關的WCAG規則和一致性層級，以及頁面上受影響的元素。 若要瞭解如何解譯結果並解決機會，請參閱[預檢中的稽核結果](../audit-results.md)。
