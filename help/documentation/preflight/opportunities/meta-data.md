# 預檢後設資料機會

![預檢後設資料機會](./assets/metadata/hero.png){align="center"}

Sites Optimizer 中的預檢後設資料機會可協助您識別和解決網站上後設資料使用相關問題。 此機會聚焦於評估您網站的後設資料結構，並提供可操作的改善建議。
