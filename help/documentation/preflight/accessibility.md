# 預檢協助工具機會

![預檢協助工具機會](./assets/accessibility/hero.png){align="center"}

Sites Optimizer中的「預檢協助工具」機會可協助您識別並解決網站上的協助工具問題，確保可供所有個人使用，包括殘障人士。 此機會著重於評估網站與協助工具標準的相容性，並提供可行的改善建議。
