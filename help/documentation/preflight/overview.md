# 預檢機會

![預檢機會](./assets/overview/hero.png){align="center"}

Sites Optimizer預檢機會由一組評估組成，這些評估會在發佈網頁之前分析網站內容和結構的不同方面。 這些評估有助於識別潛在問題，並提供可操作的建議以改善網站的整體品質和效能。

## 預檢設定

請依照[預檢設定](./setup.md)指南中的步驟，在您的網站中設定Preflight擴充功能。
