# 存取預檢

預檢可使用於 AEM Sites 的製作環境中，包括通用編輯器、文件型製作及 AEM Sites 頁面編輯器。「預檢開啟」是您製造頁面上的面板，會在該頁面上[執行稽核](./audits.md)，並且[顯示任何最佳化機會](./audit-results.md)，方便在發佈前改善頁面。

* **通用編輯器**：預檢擴充功能顯示在&#x200B;**側邊欄**&#x200B;中。選取此選項以開始稽核當前頁面。
* **Document-based 製作**：透過 Sidekick (小書籤) 對預覽的頁面內容執行預檢工具，以檢視機會清單。
* **AEM Sites 頁面編輯器**：使用瀏覽器中的預檢小書籤開始稽核。

如需設定說明，請參閱[預檢設定](./setup.md)。
