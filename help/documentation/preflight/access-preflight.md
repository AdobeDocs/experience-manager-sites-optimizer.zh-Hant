# 存取預檢

Preflight可用於AEM Sites製作環境，包括通用編輯器、檔案式製作和AEM Sites頁面編輯器。 預檢會以您正在編寫的頁面上的面板開啟。 從面板，您[在該頁面上執行稽核](./audits.md)，預檢[會顯示它找到的任何機會](./audit-results.md)，以在發佈之前增強頁面。

* **通用編輯器**：預檢擴充功能顯示在&#x200B;**側邊欄**&#x200B;中。 選取此選項以開啟「預檢」面板。
* **檔案式製作** — 從Sidekick開啟「預檢」面板，或開啟預覽頁面內容上的書籤小程式。
* **AEM Sites頁面編輯器** — 使用瀏覽器中的預檢書籤小程式來開啟「預檢」面板。

面板開啟後，選取&#x200B;**分析頁面**&#x200B;以開始稽核目前頁面。

如需設定說明，請參閱[預檢設定](./setup.md)。
